# GoodShyt Memory

Context continuity, relational memory, and trust-preserving session intelligence for human-centered systems.

## Features
- user profile memory
- session note capture
- memory graph links
- trust levels and tags
- FastAPI endpoints for add, recall, and summarize

## Quickstart
```bash
pip install -e .[dev]
uvicorn goodshyt_memory.api:app --reload
```
